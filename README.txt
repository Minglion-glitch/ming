 项目结构

  pythonGUI/
  ├── main.py          # 程序入口，运行 python main.py 启动
  ├── game_engine.py   # 游戏引擎：棋盘管理、落子验证、胜负判定
  ├── ai_player.py     # AI模块：三种难度（简单/普通/困难）
  └── gui_app.py       # GUI界面：棋盘绘制、控件、事件、主题、排行榜

  提高功能（选做，已实现6项）

  1. 难度选择 — OptionMenu 切换「简单/普通/困难」
  2. 计分系统 + 最高分 — JSON文件持久化
  3. 暂停/继续 — 不破坏游戏状态
  4. 游戏结束弹窗 — messagebox 统计信息 +「再来一局」
  5. 主题切换 — 经典/海洋/森林 三套配色
  6. 排行榜 — Top 5 历史最高分，Treeview展示

  拓展功能（已实现1项）

  - 人机对战AI — 简单（随机）、普通（评估函数）、困难（Minimax + Alpha-Beta剪枝，搜索深度2层）

  运行方式

  cd aim文档
  python main.py

  黑子为玩家（先手），白子为AI。点击棋盘交叉点落子，AI会自动回应。
